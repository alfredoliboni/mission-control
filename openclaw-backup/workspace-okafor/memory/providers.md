# Providers - Ottawa Area

Last Updated: 2026-04-07
Status: Working List

## Use Notes
- This file is a practical shortlist, not a verified referral directory.
- Verify intake status, age ranges, and current wait times before recommending action.
- Prioritize providers who can work with co-occurring ASD and ADHD rather than treating attention issues as simple noncompliance.

## Hospital / Public System
### CHEO
- Type: Pediatric hospital and specialty services
- Region: Ottawa
- Relevance:
  - developmental and behavioural pediatrics ecosystem
  - autism-related diagnostic and consult pathways
  - ADHD assessment / management support depending on clinic pathway
  - allied health and child development navigation
- Best use for Liam:
  - specialist follow-up
  - coordination where school, medication, and developmental needs overlap
- Verification needed: exact clinic entry route and current referral requirements

### OCTC (Ottawa Children’s Treatment Centre) / CHEO developmental rehab ecosystem
- Type: Children’s treatment / rehabilitation support
- Region: Ottawa
- Relevance:
  - developmental services, rehab, family navigation, and school-linked supports depending on program stream
  - can be relevant where functional needs overlap across communication, regulation, and participation
- Best use for Liam:
  - system navigation
  - therapy coordination questions
  - rehab-linked developmental supports
- Verification needed: current service map and whether program naming has changed under CHEO integration

## Autism / Behaviour / Parent Coaching
### Ottawa ABA / behaviour providers
- Type: Private autism and behaviour services
- Region: Ottawa
- Good fit criteria:
  - experience with school-age children
  - works on regulation, routines, and functional skills
  - comfortable with ADHD and executive function needs
  - does not rely on overly compliance-based models
- Search focus for family use:
  - parent coaching
  - school consultation
  - social skills support
  - behaviour consultation for transitions and regulation
- Verification needed: specific provider roster and current openings

- Type: Medical specialist
- Region: Ottawa
- Relevance:
  - medication review

  - support for school documentation
- Best use for Liam:
  - structured medication follow-up

  - integrated recommendations for ASD + ADHD

### Psychologists / neuropsychology practices with autism + ADHD experience
- Type: Private assessment / consultation
- Region: Ottawa
- Relevance:
  - diagnostic clarification if needed later

- Type: Group-based intervention or recreation support
- Region: Ottawa
- Best fit for Liam:
  - low-ratio groups
  - structured routines

- Verification needed: seasonal offerings and whether supports are clinical or recreation-based

## Recommended Intake Strategy
1. Start with school-impact needs: OT, SLP, and medication follow-up
2. Add one autism/behaviour consultation option for parent coaching
3. Prefer providers who will write practical recommendations for home and school
4. Ask every provider directly: "Do you work with children who have both autism and ADHD?"
