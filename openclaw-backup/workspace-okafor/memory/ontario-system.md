# Ontario System Reference - Ottawa Focus

Last Updated: 2026-04-07
Status: Reference

## Big Picture
Liam’s planning sits across multiple systems at once:
- autism services
- ADHD medical management
- school supports
- disability benefits
- community programming

For a child with ASD + ADHD, delays often happen when each system expects another system to carry the full load. This file is the quick map.

## Ottawa / Regional Anchors
### CHEO
- Role: key pediatric hospital and specialist hub in Ottawa
- Common relevance:
  - developmental and behavioural pediatrics
  - diagnostic follow-up
  - referrals tied to child development concerns
  - support where medication and neurodevelopment overlap
- Use for Liam:
  - specialist follow-up
  - coordination questions when school, medication, and development intersect

### OCTC
- Role: Ottawa children’s treatment / rehab reference point
- Common relevance:
  - developmental rehabilitation ecosystem
  - therapy-linked functional supports depending on stream
  - family navigation in child disability services
- Use for Liam:
  - understanding local treatment pathways
  - identifying rehab or developmental service routes
- Note: verify current branding and integration details with CHEO-era service map

## School System
### Ottawa-Carleton District School Board (OCDSB)
- Role: school board support system for accommodations and special education planning
- Relevant tools:
  - IEPs
  - school-based team meetings
  - board special education processes
  - possible IPRC if needs increase or placement/formal identification becomes important
- For Liam, priority asks may include:
  - movement breaks
  - visual schedule and transition prep
  - reduced writing/output burden where appropriate
  - seating and sensory supports
  - communication support when fluency worsens under stress
  - behaviour language that recognizes regulation and executive function needs

## Provincial Autism System
### Ontario Autism Program (OAP)
- Role: main provincial autism service pathway
- Important point: ADHD does not create a separate OAP stream, but it should be documented because it changes functional impact and support planning.

## ADHD Medical System
### Primary care / pediatrician / developmental pediatrician
- Role: medication management and monitoring
- Important point: school feedback is often essential to judge whether medication is helping in real settings.
- For Liam: medication follow-up should look at focus, rebound, appetite, sleep, and classroom participation.

## Local Service Coordination Logic
- If the question is therapy or autism funding, start with OAP and autism/developmental service contacts.
- If the question is medication or ADHD symptom management, start with the prescribing clinician or pediatric specialist pathway.
- If the question is accommodations during the school day, start with the school team and IEP process.
- If the question is respite or family support, review SSAH and local community agency options.

## Practical Ottawa Strategy
1. Keep one short summary of Liam’s dual diagnosis and daily functioning ready to reuse
2. Bring both ASD and ADHD reports to school and provider intakes
3. Ask every service whether they understand co-occurring autism and ADHD
4. Push for concrete accommodations, not generic reassurance
5. Treat school, medical, and service systems as linked, not separate
