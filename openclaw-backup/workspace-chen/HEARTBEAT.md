# Heartbeat Tasks

Every heartbeat cycle, do the following:

1. Check if any alert deadlines are approaching (within 2 weeks)
2. Search for new providers or programs in the family region
3. Check for updates to OAP waitlists or benefit programs
4. Update pathway.md with any progress
5. Write a brief summary of what you found to alerts.md if anything new

Keep updates concise. Only alert the family for actionable items.
