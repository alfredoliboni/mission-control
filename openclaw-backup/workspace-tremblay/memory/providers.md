## Priority Provider Categories to Build Out

### Speech-Language Pathology
- Need: Bilingual FR/EN or French-capable SLP preferred
- Search priority:
  1. Five Counties Children's Centre
  2. Autism-related provider directories and local provider directories
  3. College-registered private SLPs who explicitly serve children in French or bilingually
- Intake question to ask:
  - Can therapy and parent coaching be offered in French, English, or both?

### Occupational Therapy
- Need: Pediatric OT with sensory profile experience
- Search priority:
  1. Five Counties Children's Centre
  2. Local private pediatric OT clinics
  3. Providers comfortable supporting feeding texture concerns if relevant
- Bilingual note:
  - French-friendly parent coaching is especially valuable even if direct child therapy is mainly in English

### ABA / Autism Therapy Providers
- Need: Family-centred, practical support, especially early post-diagnosis
- Notes:
  - For Mia, support should focus on communication, emotional regulation, daily functioning, and parent coaching, not only compliance-based goals
  - Provider fit matters more than simply finding the first opening
- Verification status:
  - Specific Peterborough private ABA provider names still need verification before recommendation

### Developmental Pediatrics / Medical Follow-up
- Need: Follow-up for developmental questions, referrals, and benefit paperwork support
- Notes:
  - Access may occur through pediatrician / family doctor referral pathways rather than direct self-referral
  - Keep track of which physician is willing to complete forms, referrals, and school-facing documentation
- Verification status:
  - Specific Peterborough-area developmental pediatrician names still need verification before recommendation

## French-Language Provider Strategy
- Always ask whether the provider can:
  - assess in both French and English
  - coach parents in French
  - provide French handouts or home programs
  - document how bilingualism was considered in interpretation
- If direct French care is unavailable, ask for:
  - bilingual home practice recommendations
  - French-language parent sessions
  - referral onward to French-capable clinicians
