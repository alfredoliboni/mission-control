# Pathway - Mia Tremblay

Last Updated: 2026-04-07
Status: Active

## Current Stage
Early post-diagnosis pathway for a recently diagnosed child in Ontario.

## Journey Stages

### 1. Diagnosis
- Status: Completed
- Diagnosis: ASD Level 1
- Diagnosis Date: 2025-09
- Notes: Written diagnostic report should be stored and used for OAP registration, school planning, and future benefit applications.

### 2. Immediate Post-Diagnosis Organization
- Status: In Progress
- Tasks:
  - Secure digital and paper copies of diagnostic assessment
  - Track referrals, waitlists, provider contacts, and school communication
  - Confirm preferred family language for every intake

### 3. OAP Registration
- Status: Urgent, Not Yet Started
- Why next: OAP registration is a major gateway step after diagnosis.
- Tasks:
  - Create or activate AccessOAP account
  - Submit written ASD diagnosis and required supporting documents
  - Save confirmation, case details, and login information
- Notes:
  - Family should register as soon as possible.
  - Foundational Family Services may be available while waiting for other service pathways.

### 4. School Notification and Support Planning
- Status: Urgent, Not Yet Started
- Why next: School supports can often begin before outside therapy starts.
- Tasks:
  - Notify school/principal of diagnosis
  - Request school meeting
  - Request IEP discussion based on assessed needs
  - Document bilingual communication needs and any sensory/feeding concerns
- Peterborough note:
  - If in English public system, likely KPRDSB.
  - If French-language schooling is preferred, review Conseil scolaire options serving Peterborough.

### 5. Service Intake
- Status: Not Yet Started
- Priority areas:
  - Speech-language pathology, ideally French-capable or bilingual
  - Occupational therapy, especially sensory-related support
  - Autism-family coaching / parent support
  - Social skills supports
- Local system note:
  - Five Counties Children’s Centre is a core local pediatric rehabilitation contact point.

### 6. Benefits and Funding Setup
- Status: Not Yet Started
- Tasks:
  - Review Disability Tax Credit application pathway
  - If DTC approved, follow with Child Disability Benefit and RDSP planning
  - Track OAP-related timelines and respite options

### 7. Family Support and Stabilization
- Status: Not Yet Started
- Tasks:
  - Connect with autism family support resources
  - Find French-language parent resources where possible
  - Build a realistic 3 to 6 month action plan instead of trying to solve everything at once

## Recommended Order of Operations
1. Organize diagnosis documents
2. Register for OAP
3. Notify school and request support meeting / IEP discussion
4. Start local service intakes
5. Start DTC and related benefits process
6. Add family support and respite options
